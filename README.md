# Event Management System

## 📌 Project Description
This is a web-based Event Management System developed using Java Servlets and JSP. It allows users to register, login, and manage event bookings efficiently.

## 🚀 Technologies Used
- Java (Servlets)
- JSP
- HTML, CSS
- MySQL

## 🔑 Features
- User Registration & Login
- Event Booking
- Event Management
- Database Integration

## 🏗️ Architecture
Follows MVC architecture:
- Model: MySQL
- View: JSP
- Controller: Servlets

## 💡 Future Improvements
- Convert to Spring Boot
- Add REST APIs
- Improve UI design
